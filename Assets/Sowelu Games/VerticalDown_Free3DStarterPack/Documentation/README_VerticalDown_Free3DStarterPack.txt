Vertical Down Free 3D Starter Pack
Version 1.0
Created by SoweluGames

DESCRIPTION
Vertical Down Free 3D Starter Pack is a simple stylized 3D asset pack created for Unity beginners, students, and developers building vertical platformer prototypes.

INCLUDED CONTENT
- 1 player character model
- 4 platform prefabs
- 2 obstacle prefabs
- 2 collectible prefabs
- 1 decorative broken totem prop
- 1 demo scene
- Materials and textures
- Skybox material and texture

HOW TO USE
1. Import the package into your Unity project.
2. Open the demo scene:
   Assets/Sowelu Games/VerticalDown_Free3DStarterPack/Scenes/
3. Use the prefabs located in:
   Assets/Sowelu Games/VerticalDown_Free3DStarterPack/Prefabs/

PREFAB STRUCTURE
Each prefab uses a clean root object with the visual mesh as a child.

Example:
PFB_VD_Platform_Square_01
- Box Collider on root
- Mesh model as child

SCALE
Recommended scale:
1 Unity unit = approximately 1 meter.

COLLIDERS
- Platforms use colliders for gameplay.
- Collectibles use trigger colliders.
- Obstacles use trigger colliders.
- Decorative props may have optional colliders.

SCRIPTS
This package does not include gameplay scripts.

AI-ASSISTED CONTENT DISCLOSURE
Some 3D assets in this package were generated using AI-assisted tools and then manually prepared, scaled, material-adjusted, organized, and configured as Unity prefabs by SoweluGames.
